大多數人用 AI 的方式，大概就是打開 ChatGPT 或 Claude，問個問題，得到答案，複製貼上，結束。

進階一點的，可能會請它幫忙寫信、翻譯、整理重點。但說到底，我們跟 AI 的互動模式還是：**你問，它答**。

不知道大家有沒有想過，如果 AI 不只是「回答你的問題」，而是像同事一樣，你交代一件事，它自己去搞定呢？

Anthropic 在 2026 年初，在 Claude Desktop 上推出了 **Cowork** 這個功能，就是在做這件事。今天來簡單介紹一下：

* Cowork 是什麼
* 它能做哪些事
* Skills 又是什麼
* 一些使用上的注意事項

## Cowork 是什麼

簡單來說，Cowork 就像是你在 Claude Desktop 裡多了一位**數位同事**。

過去用 Claude 就是一來一回的聊天，你打一句，它回一句。但切換到 Cowork 模式後，你可以交代一整個任務，然後放著讓它做。等你泡完咖啡回來，事情就做好了。

用過 Claude Code 的工程師應該會很熟悉這種感覺 — 交代一件事，Claude 自己去讀程式碼、寫檔案、跑測試。Cowork 本質上就是把同樣的能力，包在一般人友善的介面裡。你不需要會寫程式，不需要打指令，只要用白話文描述你想要的結果就好。

Anthropic 自己的說法是：

> Cowork lets you complete non-technical tasks much like how developers use Claude Code.

嗯，就是 **Claude Code for the rest of your work**。

## 它能做什麼

Cowork 可以直接存取你電腦上的檔案（當然，你要先授權哪些資料夾它可以動），然後在上面做各種操作。

舉幾個情境：

### 整理檔案

你的「下載」資料夾是不是已經亂到不想打開了？😂 你可以跟 Cowork 說「幫我把下載資料夾裡的檔案按照類型跟日期整理好」，它就會自己去分類、重新命名、建資料夾，一次搞定。

### 做研究彙整

假設你在準備一個專案，需要研究某個主題，你可以請 Cowork 去搜集資料、整理成一份重點摘要，直接存成文件放在你指定的資料夾裡。不用自己開十幾個分頁、複製貼上、再手動排版。

### 製作文件

從散亂的筆記生成一份簡報、從一堆收據截圖整理成有公式的 Excel 報表、把會議逐字稿抓出重點和待辦事項 — 這些它都能做。而且產出的不是純文字，是真正格式完整的 `.xlsx`、`.pptx`、`.docx` 檔案。

### 平行處理多個任務

跟真正的同事一樣，你可以一次交代好幾件事。Cowork 會自己把任務拆成多個子任務，用多個 sub-agent 平行處理。你不需要排隊等一件做完才交代下一件。

它整個執行過程是跑在**虛擬機（VM）**裡的，跟你的系統有一定程度的隔離，所以相對安全。而且過程中你隨時可以看到它在做什麼、打算怎麼做，覺得方向不對也可以隨時介入調整。

## Skills 是什麼

如果說 Cowork 是你的數位同事，那 **Skills** 就像是這位同事具備的「專業技能」。

Anthropic 替 Cowork 準備了一系列的 Skills，讓它在處理特定類型的任務時更得心應手。比如它有處理 Excel、PowerPoint、Word、PDF 這些辦公室常見格式的 Skills，所以它不只是把文字塞進檔案裡，而是能理解檔案的結構，產出有 VLOOKUP 公式的試算表、有排版的簡報。

Anthropic 也開源了 11 個 Plugins（裡面就包含各種 Skills），涵蓋了銷售、法務、行銷、客服等不同領域。

你可以把它想成：**Cowork 本身是一個聰明的通才，而 Skills 讓它在特定領域變成專家。**

## 使用上的注意事項

Cowork 目前還是 Research Preview 的階段，有幾件事值得留意：

- **需要付費訂閱**：Pro（$20/月）、Max、Team 或 Enterprise 方案才能使用
- **比聊天更吃額度**：多步驟的任務需要比較多的運算資源，用量會消耗得比一般聊天快不少
- **僅限桌面版**：目前只有 macOS 和 Windows 的 Claude Desktop 可以用，手機跟網頁版還沒支援
- **沒有跨 session 記憶**：每次新的 Cowork session 是獨立的，不會記得上一次的對話內容

## 我的看法

我覺得 Cowork 代表了一個蠻明確的趨勢：**AI 的使用方式正在從「對話」走向「委派」**。

以前我們跟 AI 互動，是一步一步地引導它、複製貼上結果、手動整合。但現在你可以直接說「我要什麼結果」，然後讓它自己去規劃、執行、交付。

我們太習慣把 AI 當成一個比較厲害的搜尋引擎或翻譯機了。但 Cowork 這類工具的出現，其實是在提醒我們 — AI 能做的事情，比我們以為的還要多很多。

如果你已經有在用 Claude 或 ChatGPT 了，不妨試試看 Cowork，體驗一下從「問問題」到「交任務」的差別，說不定會打開一些新的想法～
